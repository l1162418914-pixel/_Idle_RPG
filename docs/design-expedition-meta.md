# 出征元系统（1/7 与编队）· 讨论定稿

> **状态：已实现（核心）** — 双半组、养伤锁、基地编队 UI、自动补员；战间替补上阵已落地。

## 挂机 A+B

| 功能 | 行为 |
|------|------|
| **再次出征** | 结算领奖 → 同地图 → 上一趟编队快照 + 自动补员 → `start_run` |
| **自动连续出征** | 勾选后结算可倒计时自动循环；失败/养伤锁时停止 |
| **自动补员** | 空位从名册按规则补满；不跨半组抢人；替补槽允许空 |
| **地图** | 再战不刷新 `selected_map_id` / `auto_run_map_id` |

## 双半组 4+2（H2 仅基地换班）

| 项目 | 规则 |
|------|------|
| **编制** | 半组 A/B 各 **4 出战（主角+3 佣兵）+ 2 替补（可空）**；同一人不可同时在 A、B |
| **本趟** | **不**在半路 A↔B 换班 |
| **回城** | 优先 **A** 出征；A 四出战不可 `can_join_squad()` → 自动 **B**；地图不变 |
| **组内替补** | 撤离 / 双人濒死可顶替补（组内缓冲）；整组不可用再走半组切换（仅 BASE） |

## 全队养伤锁

| 项目 | 规则 |
|------|------|
| **触发** | A、B 两半组 **4 出战位均不可出征** |
| **期间** | 禁再次出征/自动循环；**养伤 UI**；地图 ID 保留 |
| **解除** | **任一半组** 四出战全员 `can_join_squad()`（**70% HP** 清濒死，见 near-death 文档） |
| **恢复** | 基地慢回；养伤中 **医疗室加速**；医护 **优先治疗最快能满队的一组** |

## 出征失败提示

- `start_run` 返回 -1/-2/-3 时基地/编队 UI 显示 `get_run_start_error_message`。
