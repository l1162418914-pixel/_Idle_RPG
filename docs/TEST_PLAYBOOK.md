## 1. 界面速查（看哪里）

### 大营 BASE

| 区域 | 看什么 |
|------|--------|
| 顶部 | **双半组编队**（A/B 半组、槽位、未指派池） |
| 地图列表 | 正式图在上，**测试 / 演练** 分区在下（带【测试】前缀） |
| 点地图后 | 进入 **出征准备**（Squad 界面） |

### 出征准备 PREPARE

| 行/块 | 含义 |
|------|------|
| 地图名 + 危险 + Boss 距离 | 例如 `Boss 90.0m` |
| 团队稳定度 | ≤30 时可能显示「强制撤离」提示 |
| `[测试·xxx]` 横幅 | 来自地图 `description` |
| **测试参数:** | 阈值、格子、掉率等（无撤离点的图也有） |
| 编队已锁定 | ⑤⑥ 会写「本测试图编队已锁定…」 |

### 出征中 RUN

| UI | 何时出现 |
|----|----------|
| 顶部距离条 | 当前前进距离 / 最大距离 |
| **携带价值** | ④ 等有价值阈值的图 |
| **撤离物: xxx** | ② 捡到撤离物后 |
| **Boss 追击 · 压力%** | ③ 及正式追击图（①②④⑤⑥⑦ 测试图多数 **无**） |
| 底部浮动提示 | 返程、自动撤、守卫战、觉醒等 |
| 战斗窗 **Debug 工具栏** | 慢速/暂停日志；标签「正式数值」或「测试模式 ON」 |
| 返程中按钮 | **追击反击**、蓄力击退、**深度反击**（③） |

### 结算 RESULT

| 看项 | 说明 |
|------|------|
| 胜/败/撤离 | 是否「全军覆没」 |
| 返程原因 | `forced` / `auto_value` / `manual` 等 |
| **再战** | 同图重开（保留测试编队规则） |

### ① 稳定度返程 · `retreat_drill`

**验什么**：团队稳定 ≤30 → **强制返程**；有护盾；**无** Boss 追击条。

#### 准备页应看到

| 项 | 期望值 |
|----|--------|
| 横幅 | 含【正常】接战 1~2 波、稳定≤30、无追击 |
| 测试参数 | 无 Boss 追击；掉率约 8% |
| Boss 距离 | 320m（可不管，不必推到 Boss 区） |

#### 操作剧本

1. 选 **测试①**，半组至少 2~4 人，**出征**。
2. 进图后 **主动靠近怪并接战 1~2 波**（别一直跑）。
3. 盯 **团队稳定度**（准备页/出征 UI 若有）或等浮动提示。
4. 稳定 ≤30 后应 **自动开始返程**（仍可接战）。
5. **走回大营** → 看结算。

#### 屏幕上应对齐的文案（含近似）

| 时机 | 文案方向 |
|------|----------|
| 稳定跌破 | `稳定度 xx，建议撤离` 或 `稳定度过低，强制返程` |
| 返程开始 | `从 xx m 返程 → 大营` |
| 护盾 | `返程护盾 装备 x/x · 物资 x/x` |
| **不应出现** | `Boss 追击`、`首领开始追击` |

#### 通过 / 失败

- [ ] **通过**：强制返程 + 正常回营结算（非灭团）
- [ ] **失败**：记录：稳定是否不掉 / 是否无返程 / 是否误出追击

**误报不算 bug**：晚几秒返程、多打一波才掉稳定。

### ② 撤离物线 · `test_extract`

**验什么**：高掉率撤离物、守卫战、**55m 撤离点**。

#### 准备页应看到

| 项 | 期望值 |
|----|--------|
| 撤离点 | **55m** |
| 智能撤离阈值 | **200** |
| 撤离物掉率 | **32%** |
| 测试参数 | 无 Boss 追击 |

#### 操作剧本

1. 出征，杀 **3~6 只** 怪（推进稍快 `advance 2.2x`）。
2. 看距离条是否出现 **撤离物: …**。
3. **拾取**后两种分支：
   - **A**：提示 **宝库守卫战** → 打赢 → **直接结算**（通关式）。
   - **B**：提示 **未引动守卫，已占格** → 手动 **返程** → 应先朝 **55m 撤离点** → 再大营。

#### 应对齐的文案

| 分支 | 文案方向 |
|------|----------|
| 守卫 | `拾取 xxx：触发宝库守卫战！` |
| 安全拾取 | `拾取 xxx：未引动守卫，已占格` |
| 到撤离点 | `已抵达撤离点，继续返回大营…` |
| 返程物线 | `撤离物线：返程刷怪加密…`（若已触发守卫线） |

- [ ] **通过**
- [ ] **失败**：记是否不掉撤离物 / 守卫战未触发 / 返程不经过 55m

### ③ Boss 追击 · `test_boss_chase`

**验什么**：~90m 区域首领 → **自动返程勿杀**；追击条；反击/蓄力/深度 **不通关**；**杀追击首领 = 通关**。

#### 准备页应看到

| 项 | 期望值 |
|----|--------|
| Boss 距离 | **90m** |
| 测试参数 | **区域首领出现即自动返程**；推进快 |
| 横幅 | 勿杀区域首领、深度反击≠通关 |

#### 操作剧本

**阶段 A — 触发自动返程**

1. 出征，**快速推进**（约 2.8 倍速）到 **~85–95m**。
2. 等 **区域首领** 出现 → **不要击杀**。
3. 应提示 **测试图自动返程** 并开始返程。

**阶段 B — 追击**

4. 确认顶部 **Boss 追击 · 压力%**。
5. 被追上进入僵持 → 试 **蓄力击退**（满蓄松开）→ 应 **继续返程，不通关**。
6. 中距离试 **追击反击** → 首领变远 + 经验提示。
7. 僵持中蓄力 **≥约 18%** → 点 **深度反击** → 重创推远，**仍不通关**。

**阶段 C — 通关**

8. （可选另开一局）追击战中 **击杀追击首领** → 应 **直接通关结算**。

#### 应对齐的文案

| 时机 | 文案方向 |
|------|----------|
| 自动返程 | `测试图自动返程` 类 |
| 追击开始 | `首领开始追击！` |
| 反击 | `反击成功！首领被推远…` |
| 深度 | `深度反击额外经验…` |
| 击杀追击 Boss | `追击首领被击杀！` → 通关 |

- [ ] **通过**
- [ ] **失败**：记是否杀了区域首领导致异常 / 无追击条 / 深度反击直接通关

### ④ 携带价值撤离 · `test_auto_value`

**验什么**：携带价值 **≥140** → `auto_value` 自动返程。

#### 准备页应看到

| 项 | 期望值 |
|----|--------|
| 携带价值阈值 | **140** |
| 装备掉率 | **42%** |
| 测试参数 | 无 Boss 追击 |

#### 操作剧本

1. 出征，杀怪拾取 **装备**（多打几只）。
2. 盯距离条 **携带价值** 上升。
3. 达到 **140** 前应提示 **`携带价值 xx≥140，自动返程`** 或类似。
4. （对比）勾选出征 **「自动撤:仅安全箱」** 再测一局 → 应 **更难** 触发（外露不计）。

- [ ] **通过**（正式计数）
- [ ] **对比**（仅安全箱）已测

### ④b 网格满撤 · `test_loot_full`

**验什么**：**不靠价值**（阈值 9999），靠 **格子满** → `auto_rule` 返程。

#### 准备页应看到

| 项 | 期望值 |
|----|--------|
| 携带价值阈值 | **9999**（不应靠价值触发） |
| 外露格 | **3×2** |
| 掉率 | **38%** |

#### 操作剧本

1. 出征，连续杀怪捡装备直到 **安全箱或外露格快满**。
2. 应提示 **`自动规则触发返程`** 或 `携带价值…` 但 reason 为 **auto_rule**（看浮动字）。
3. 确认 **不是因为** 价值到 140（阈值是 9999）。

- [ ] **通过**
- [ ] **失败**：记是价值触发还是始终不满

### ⑤ 单人濒死 · `test_near_death_solo`

**验什么**：**仅主角**；稳定骤降返程；HP0=**濒死**；可 **紧急撤离**。

#### 准备页应看到

| 项 | 期望值 |
|----|--------|
| 编队 | **仅主角**，名册不可改（锁定） |
| 横幅 | 单人濒死说明 |

#### 操作剧本

1. 确认只有主角出战 → 出征。
2. 接战至 **强制返程**（稳定机制同①，衰减更快）。
3. 返程中或接战让主角 **HP 归零** → 状态 **濒死**（非立即永久死亡）。
4. 使用 **紧急撤离** 回大营。
5. 大营养伤至 **≥70% HP** 后再出征。

- [ ] **通过**
- [ ] **失败**：记是否多人出场 / 濒死变秒死

### ⑥ 双人濒死 · `test_near_death_duo`（T-02e）

**验什么**：**主角 + 1 佣兵**；单趟 **3~5 min**；返程 **≥2 波**；便于濒死/后排站位（T-02a）。

#### 准备页

| 项 | 期望值 |
|----|--------|
| 测试档 | `python tools/build_test_save.py` 生成 slot1；主角 Lv17、搭档佣兵 Lv16 |
| 名册 | 锁定 **2 人** |
| 用时 | 秒表：点出征 → 回营结算 **180–300s** |

#### 操作剧本

1. 选 **测试⑥**；前排槽放易挨打单位。
2. 去程接战 **多波**（非 1–2 波即返程）→ 稳定度强制返程。
3. 返程途中 **≥2 波** 接战；尽量让 **两人** 都进濒死。
4. 回营结算，记秒表。

- [ ] **通过**
- [ ] **失败**：记实际秒数 / 是否仍碾压

#### 准备页

|----|--------|
| awakening | 配置 **0.5** 概率 |

#### 操作剧本

2. 约一半局数应出现 **`xxx 绝境觉醒·爆发/盾援/铁壁/回光`**。
3. 确认 **无** Boss 追击 UI。

- [ ] **通过**（至少见过 1 次觉醒）
- [ ] **失败**：记从未触发 / 每把都触发
