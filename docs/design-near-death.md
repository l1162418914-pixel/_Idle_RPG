# 5/7 濒死系统 · 讨论定稿

> **状态：已实现（核心）** — 濒死/70%、伤痕、搀扶、医疗室消伤痕、组内替补；**绝境觉醒**含表驱动变体（爆发/盾援/铁壁/回光）。

## 一、基础（保留/修订）

| 项目 | 现设计 / 修订 |
|------|----------------|
| 触发 | 友方 HP→0 → `is_near_death` + `DOWNED`，非永久死亡 |
| 永久死亡 | 仅撤离失败等 `mark_permanent_death` |
| 出征 | `can_join_squad()` 需非濒死；回城 **≥70% max_hp** 清濒死（`try_clear_near_death_for_deploy`） |
| 返程移速 | 任一队员濒死 ×0.5；与搀扶惩罚 **叠乘** |
| 战斗 | 濒死不攻击、不移动（觉醒窗口除外） |

---

## 二、A. 搀扶（自动）

| 项目 | 规则 |
|------|------|
| 触发 | **自动**为濒死者分配搀扶者（建议 1:1，出战 4 人内） |
| 搀扶者 | **可正常攻击**、可参战 |
| 代价 | 存在需搀扶濒死者时 **全队**返程移速略减（如 -12%，与 ×0.5 叠乘） |
| 被扶者 | 维持濒死；不永久死亡/不移除编制 |
| 实现 | `supported_by_id` 等标记；可不单独 ActionState |

---

## 三、B. 伤痕 debuff

| 项目 | 建议 |
|------|------|
| 累积 | 每次 **进入濒死** +1 层（或仅抵营仍濒死 +1，实现时择一） |
| 上限 | `scar_stacks` cap 5~10 |
| 效果 | 表驱动（`near_death_config.json` → `scars`）：生命/攻击倍率、受伤稳定度加压 |
| 消除 | 基地 **医疗室**：金币 + 时间（或加倍金币立即）；与养伤（70% HP）**分开** |
| 出征 | 建议 **可带伤出征**（有伤痕但已满足 70% 与 `can_join_squad`） |
| 觉醒 | 建议不因觉醒多叠层（仍按「进濒死」计） |

---

## 四、C. 绝境觉醒

| 项目 | 规则 |
|------|------|
| 基础概率 | **3%~5%**（检定时机：实现时固定一种，如每 N 秒或受击前） |
| 加成 | 装备、技能、Buff → `awaken_chance` / 持续 / 伤害倍率 |
| 触发后 | 进入 **`AWAKENING`** 状态 **T 秒**（建议 3~8s） |
| 期间 | **高输出**（攻倍、攻速或一次技能）；可被选为目标；仍低血 |
| 结束 | 回到 **濒死 / DOWNED**（HP 低位，如 5%~15%） |
| 上限 | 建议 **每名每趟最多 1 次**（或 CD） |
| 搀扶 | 觉醒期间可不扶；结束后自动再扶 |

### 觉醒变体（表驱动，首期可做一种）

| id | 效果 |
|----|------|
| `damage_burst` | 高输出（**默认**） |
| `team_shield` | 短时物资/装备盾加成（见 retreat 文档） |
| `taunt` | 短时吸引仇恨 |
| `heal_snap` | 自疗后仍回濒死 |

---

## 五、与返程护盾（物资层）

- **濒死烧材料**：增加 `material_shield_current`；材料从网格删除；**不烧穿着装备**。
- 默认：**优先安全箱材料**，不足再外露。
- 装备护盾：见 [design-retreat.md](design-retreat.md) §7（开撤进 CD，受击先扣装备层）。

---

## 六、与养伤 / 双半组

| 系统 | 关系 |
|------|------|
| 养伤锁 | A+B 均不可出征 → 养伤 UI；医护优先满一队 |
| 70% 可出征 | 清濒死标记，不自动清伤痕 |
| H2 | 本趟不换半组；搀扶仅当前出战 4 人 |

## 七、实现触点

| 模块 | 内容 |
|------|------|

## 八、状态机

```
ALIVE ──HP→0──→ DOWNED (near_death)
                    │
        ┌───────────┼───────────────┐
        │ 自动搀扶   │ 3~5% 觉醒      │
        │ (全队略减速)│               │
        ▼           ▼               │
   仍 DOWNED    AWAKENING (T秒)
                    │
                    └──→ 再 DOWNED
```

抵营 → 养伤/70% → 可出征；伤痕单独医疗室处理。
