## 各图「怎样算正常」

### ① 稳定度返程 `retreat_drill`

- 进图稳定度通常接近满。
- 接战 **1~2 波** 后团队稳定 **≤30** → 提示强制返程，开始往回走。
- 返程有 **装备盾/物资盾**，**没有** Boss 追击距离条。
- 回大营结算，**不是**全军覆没。

### ② 撤离物线 `test_extract`

- 准备页：撤离点 **55m**、阈值 **200**、掉率 **32%**。
- 杀 **几只** 怪就可能掉撤离物，距离条显示 **撤离物: …**。
- 拾取后可能 **宝库守卫战** → 打赢 **直接结算**。
- 未触发守卫时手动返程：先往 **撤离点**，再到大营。

### ③ Boss 追击 `test_boss_chase`

- 推到 **~90m** 出区域首领 → **勿击杀** → 提示 **测试图自动返程**。
- 出现 **Boss 追击 · 压力%**。
- **追击反击**（中距离）→ 首领变远 + 经验。
- 被追上 **蓄力击退** 满蓄松手 → **继续返程，不通关**。
- **深度反击**（蓄力≥约18%）→ 重创推远，**不通关**。
- **击杀追击首领** → **直接通关结算**。

### ④ 携带价值撤离 `test_auto_value`

- 阈值 **140**，掉率很高。
- 杀怪拾取装备后，距离条 **携带价值** 上升。
- 达到阈值 → 提示 **携带价值达标 / auto_value** 自动返程。
- 勾选出征 **「自动撤:仅安全箱」** 后，只有安全箱计入（外露不算），应更难触发。

### ④b 网格满撤 `test_loot_full`

- 阈值故意很高（9999），**不会**靠价值触发。
- 高掉率 + 小外露格（3×2）→ 安全箱或外露 **快满** → **自动规则** 返程（`auto_rule`）。
- 与 ④ 区分：一个是 **价值**，一个是 **格子满**。

### ⑤ 单人濒死 `test_near_death_solo`

- 准备页 **仅主角**，名册不可点。
- 稳定度骤降 → 强制返程（同①类）。
- 主角 HP→0 → **濒死**，非立刻永久死亡；可 **紧急撤离**。
- 回营养到 **≥70% HP** 可再出征。

### ⑥ 双人濒死 `test_near_death_duo`（T-02e 平衡）

- 使用 **`tools/seed_saves`** 测试档（约 Lv16–17 + 精良双件）；敌人 **Lv17–20**。
- 准备页 **主角 + 1 佣兵**（前排槽便于 T-02a 濒死观察）。
- 单趟 **约 3~5 分钟**（出征→稳定返程→返程 **≥2 波** 接战→回营）。
- 去程多波后再返程，可尝试两人都打濒死；非秒结束碾压。

### ⑦ 绝境觉醒 `test_awakening`

- 弱怪、**无追击**。
- 多次濒死约 **一半** 出现觉醒提示（爆发/盾援/铁壁/回光）。
- **不是** 每次都觉醒，也 **不是** 从不觉醒。

## 不算 bug 的情况

- 快慢差几秒、多刷一只怪、文案略丑。
- 关键看 **机制对不对**：该返程返程、该通关通关、该锁编队锁编队。
